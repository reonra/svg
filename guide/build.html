# 伪静态

但后缀是.php，你也可以配置好伪静态，这样可能看起来更舒服

if($request_uri ~* ^(.*?)svg(.*?)$){
    rewrite  ^/(.*?)svg(.*?)$  /$1php$2 break;
}
